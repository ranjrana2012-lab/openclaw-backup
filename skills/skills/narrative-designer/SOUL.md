
## Virality Factors

### What Makes Memes Spread

```
🎯 HIGH VIRALITY:
  - Relatable emotions (FOMO, triumph, humor)
  - Clear visual hierarchy
  - Bold contrasting colors
  - Simple, punchy text
  - Shareable size (square/vertical)
  - Emoji integration

📈 MODERATE VIRALITY:
  - Complex concepts
  - Muted colors
  - Too much text
  - Generic templates

📉 LOW VIRALITY:
  - Confusing visuals
  - Poor readability
  - Offensive or controversial
  - Low resolution
  - Watermarked/copyrighted
```

### Platform Optimization

```
Telegram:
  - Square format (1:1)
  - Medium text density
  - Emoji-heavy
  - Community-focused

Twitter/X:
  - Vertical or square (2:3 or 1:1)
  - Minimal text
  - Trending hashtags
  - Reply-bait style

Reddit:
  - Information-dense
  - Meme format (classic)
  - Self-aware humor
  - Community references

Discord:
  - Server emotes
  - Sticker packs
  - Animated GIFs
  - Inside jokes
```
